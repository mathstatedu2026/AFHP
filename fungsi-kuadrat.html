Fungsi Kuadrat

Pelajari grafik parabola, vertex, sumbu simetri, dan diskriminan dengan visualisasi interaktif

Pengertian Fungsi Kuadrat

Definisi

Fungsi kuadrat adalah fungsi polinom derajat dua yang grafiknya berbentuk parabola. Bentuk umumnya adalah f(x) = ax² + bx + c dengan a ≠ 0.

Komponen

  • a = koefisien x² (menentukan arah parabola)
  • b = koefisien x
  • c = konstanta (titik potong sumbu y)
  • Vertex = titik puncak parabola

Sifat Parabola

Jika a > 0, parabola terbuka ke atas (minimum). Jika a < 0, parabola terbuka ke bawah (maksimum).

Contoh Soal & Pembahasan

Contoh 1 Mudah

Soal: Tentukan vertex dan sumbu simetri dari fungsi y = x² - 4x + 3

Penyelesaian:

  • Diketahui: a = 1, b = -4, c = 3
  • Sumbu simetri: x = -b/2a = -(-4)/(2·1) = 2
  • Nilai y saat x = 2: y = (2)² - 4(2) + 3 = 4 - 8 + 3 = -1
  • Vertex: (2, -1)
  • Parabola terbuka ke atas (a > 0)
Contoh 2 Sedang

Soal: Tentukan akar-akar dari fungsi y = x² - 5x + 6

Penyelesaian:

  1. Hitung diskriminan: D = b² - 4ac = (-5)² - 4(1)(6) = 25 - 24 = 1
  2. Karena D > 0, ada 2 akar real berbeda
  3. x₁ = (-b + √D) / 2a = (5 + 1) / 2 = 3
  4. x₂ = (-b - √D) / 2a = (5 - 1) / 2 = 2
  5. Akar-akarnya: x = 2 dan x = 3
Contoh 3 Sulit

Soal: Tentukan persamaan fungsi kuadrat yang melalui titik (1, 0), (3, 0), dan (0, -3)

Penyelesaian:

  1. Akar-akarnya x₁ = 1 dan x₂ = 3
  2. Gunakan bentuk faktor: y = a(x - x₁)(x - x₂) = a(x - 1)(x - 3)
  3. Substitusi titik (0, -3): -3 = a(0 - 1)(0 - 3) = a(-1)(-3) = 3a
  4. Diperoleh a = -1
  5. y = -1(x - 1)(x - 3) = -1(x² - 4x + 3)
  6. y = -x² + 4x - 3
